# Markdown: Links und Bilder

Diese Datei zeigt zwei einfache Markdown-Bausteine: einen Link und ein Bild.

## Link einfügen

Syntax:

[GitHub Dokumentation](https://docs.github.com/)

## Bild einfügen

Syntax:

![Beschreibung des Bildes](../assets/beispielbild.png)

## Erklärung

Ein Link besteht aus:

- sichtbarem Linktext in eckigen Klammern
- Ziel-URL in runden Klammern

Ein Bild funktioniert ähnlich wie ein Link, beginnt aber mit einem Ausrufezeichen.

## Merksatz

Link:

[Text](URL)

Bild:

![Alt-Text](Pfad-zum-Bild)
